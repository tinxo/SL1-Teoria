## Sesión 7 - POO con C#

### Consigna desarrollada en clase

Una empresa requiere gestionar la operatoria de sus "Productos" donde se requiere manejar los siguientes datos:
- ID del producto
- Nombre
- Descripción
- Precio unitario
- Stock mínimo
- Stock actual

Como reglas de negocio se pueden mencionar:
- El stock (ambos) no puede ser menor a cero.
- El stock inicial de un producto va a ser establecido a cero.
- El precio unitario no va a poder ser menor a cero.

Se debe generar:
- La clase para administrar los productos.
- Atributos de la clase (privados).
- Propiedades (establecer lectura/escritura según corresponda) para cada atributo.
- Métodos para la gestión del stock.
